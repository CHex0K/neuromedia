  `tribe_dir / "gigaam_openrouter_corrected_words.tsv"`.
- The hybrid pipeline writes the canonical transcript to:
  `tribe_dir / "tribe_transcript.tsv"`.
- It writes the detailed corrected words table to:
  `tribe_dir / "hybrid_transcription" / "gigaam_openrouter_corrected_words.tsv"`.
- Therefore Template_3 generation passed `words_tsv=None` for normal hybrid runs.
